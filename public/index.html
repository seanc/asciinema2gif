asciinema2gif

Source Author

Asciinema2gif (asciinema to gif) is a simple website that allows you to convert your asciinema casts into gifs for use in documentation, demonstrations and various other areas. It acts as a CDN so you can just use the URL wherever you'd like.

Documentation

Asciinema2gif is very simple to use, for basic usage the URL is structured like so: https://asciinema2gif.host/gif/id

This will generate a gif from default parameters. If you would like to change the size, theme, speed, etc you should pass them as query parameters

Parameter Description Value Default
size how large should the gif be small, medium, big medium
speed how fast should the gif go between numbers 1 and 7 1
theme what theme should it used asciinema, tango, solarized-dark, solarized-light, monokai asciinema

Caveats

Conversion Time

Unfortunately you may experience slow conversion times, asciinema2gif uses the asciinema2gif cli app in the background and is relatively intensive so conversions are ran throuhh a queue.


Varying Sizes

Although you may think the sizes are universal throughount, the way asciinema records and crops is based on the window size at recording time. This is not the same for everyone and therefore the sizes are relative to the actual asciicast size.
